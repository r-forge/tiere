Political Representation in R

The R package tiere is based on George Orwell's book Animal Farm. This is a test package for R-Forge.

The package is under development and new functions will be added (i.e. the package is not feature complete at this stage). All the functions included should be fairly useable and stable.

The R package tiere is open source distributed under the MIT License. It is written and maintained by Didier Ruedin.
